# Architecture of Library Management System

This project is based on Object-Oriented Programming (OOP).

## Components:
- User: Interacts with the system
- Book: Stores book details like ID, title, author
- Library: Manages all books

## Working:
1. User selects an option (Add, View, Issue, Return)
2. System processes the request
3. Data is stored using ArrayList
4. Output is displayed to the user

## Design:
- Modular structure using classes
- Menu-driven or UI-based interaction
